# Proyector Móvil

# Introducción

# ¿Qué es?

Proyector Móvil es una aplicación de Código Abierto para computadora con el que se puede emular la funcionalidad de un proyector, emitiendo las imágenes directamente a la pantalla de los teléfonos.

# ¿Cómo funciona?

  • Los smartphone reciben las imágenes que emite la computadora por medio de la aplicación.
  • Solo el profesor puede cambiar la imagen que se ve.
  • Los usuarios se conectan a la aplicación mediante un Código QR generado por la misma.
