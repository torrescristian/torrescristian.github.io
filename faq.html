# Soluciones a problemas frecuentes

# Me olvidé la contraseña / Quiero cambiar mi contraseña / Un alumno sabe mi contraseña

  • La solución es cambiar la contraseña como se vió aquí

# La aplicación no se inicia en mi computadora

  • Puede que tenga una versión incompatible con su sistema operativo.
    Ejemplo: su sistema operativo es 32bits y descargo la versión para 64bits.

# Mi teléfono detecta el Código QR pero la página no se carga.

  • Asegurese de que el servidor esté prendido

  • Asegurese de estar conectado a la misma red Wi-Fi que la aplicación

# Descargo la aplicación pero aparece en blanco y no se abre

  • Primero debe descomprimir el programa, para ello necesita instalar winrar, puede conseguirlo de manera gratuita en https://www.rarlab.com/download.htm
