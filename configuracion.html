# Renombrar imágenes

Se deben enumerar las imágenes por orden de presentación, por ejemplo, las imágenes:

  • introduccion.jpg
  • cuadro.png
  • final.gif

Deben ser renombradas por:

  • 1.jpg
  • 2.png
  • 3.gif

WARNING

Cuide de no eliminar la extensión (.jpg, .png, .gif).

TIP

Se recomienda que las imágenes tengan un formato 16:9 (rectangular, como las pantallas de los teléfonos) esto se logra con resoluciones de, por ejemplo:
(Ancho x Alto)
1920 x 1080 pixeles
1280 x 720 pixeles

# Mover las imágenes

Mover las imágenes a la carpeta public/img

.
└─ proyector-movil
    └─ public
        └─ img <--
            ├─ 1.jgp
            ├─ 2.png
            └─ 3.gif

# Configurar el usuario del profesor

Abrir el archivo user.json con el bloc de notas (no usar otro editor)

.
└─ proyector-movil
    ├─ public
    └─ user.json <--

{
    "email": "admin@admin.com",
    "password": "admin"
}

Cambiar el email y password (contraseña) cuidando de mantener las comillas dobles

{
    "email": "elprofe@yeah.com",
    "password": "supercalifragilisticoespialidoso"
}

WARNING

No debe ingresar la misma contraseña que se usa para cuentas reales.
