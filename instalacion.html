# Instalación

# 1º Conectarse a una red Wi-Fi

Conectar todos los dispositivos a una misma red Wi-Fi que puede ser generada por un router, una notebook / netbook, o por un smartphone (No se necesita conexión a internet.)

TIP

Este puede ser un paso abrumador para varias personas, se recomienda pedir soporte a otro profesor con conocimientos técnicos, o mejor aún, a los mismos alumnos.

# 2º Iniciar la aplicación

Iniciar el programa proyector-movil.exe

.
└─ proyector-movil
    ├─ public
    ├─ proyector-movil.exe <--
    └─ user.json

El cual iniciará el servidor

WARNING

No cerrar el servidor hasta que termine la presentación.
La aplicación necesita del servidor para funcionar.

TIP

Si desea cambiar las imágenes antes debe cerrar el servidor

Y en unos segundos debería abrir una nueva pestaña en su navegador con el contenido del programa.

# 3º Leer el QR

Apuntar las cámaras al código QR que aparece en la página de inicio al abrir el proyector-movil.exe

De no contar con lector de QR se puede escribir manualmente la dirección web en el navegador, por ejemplo:

http://192.168.1.9:3000

WARNING

No olvidar el http://

# 4º Registrarse

Para ingresar a la sección Profesor y poder cambiar las imágenes debe iniciar una sesión desde Registrarse, luego de ingresar los datos será automáticamente transferido a la sección Profesor donde con las teclas de flechas puede cambiar la imagen que se visualiza, las imágenes irán apareciendo en el orden en que las nombró.

Por defecto el usuario que trae la aplicación es

{
    "email": "admin@admin.com",
    "password": "admin"
}

Para cambiar el usuario vea la siguiente sección Configuración

TIP

Tanto en los teléfonos como en las computadoras puede ver las imágenes en pantalla completa haciendo doble click sobre ellas.
